提示信息

  1. 如果需要新增模板，请将模板文件上传至mode/area/themes下
  2. 频道创建后，需要手动更新一下频道静态页
  3. 英文别名只允许英文字母和数字
  4. 对于已经创建的频道模板路径是：html/channel/{频道别名}
  5. 删除频道之后，相应的数据均会被删除
  6. 可以通过创建一个blank风格的频道来制作新的风格
